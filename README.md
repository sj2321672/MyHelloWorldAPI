# MyHelloWorldAPI

這是一個GET API，會回傳一個Hello World字串

## 如何使用

### 環境要求
- [.NET Core SDK](https://dotnet.microsoft.com/zh-cn/download) (.NET 8.0)

### 安裝專案
1. Clone專案至你的電腦：
   ```
   Git Bash
   git clone https://github.com/sj2321672/MyHelloWorldAPI.git   
   ```

### API
API將運行在http://localhost:4000/HelloWorld
- **GET** `/HelloWorld`
  返回"Hello World"字串
